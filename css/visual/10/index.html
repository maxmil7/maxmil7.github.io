10. Flex

dislpay: flex is used to control the flow of items within a parent container.
A flex container has a start and an end and two main axis around which elements are laid out (main axis, cross axis)
In row flow, horizontal axis is main axis. In column flow, vertical axis is main axis.
Parent can have multiple rows and columns
Properties are available on both parent and children to control the flow

10.1. Flex container properties

  • flex-direction: sets direction of flow of items
  • flex-wrap: determines if elements are wrapped when parent container runs out of space
  • flex-flow: shorthand for direction and flow
  • justify-content: controls gaps between items along the main axis. 'strech' value not supported in flex layout
  • align-content: controls gaps between items along the cross axis When there is only one line, align-content has no effect.
  • align-items: controls alignment of items along the cross axis
  • justify-items: Not supported/needed in flexbox. Use above properties for alignment. Flexbox specifies how content flows within it as opposed to a grid which divides the page into multiple regions.

10.1. Flex children properties

  • flex-basis: sets initial main size of flex item. Defaults to content-box
  • flex-grow, flex-shrink: sets how a flex item will grow or shrink to fit the space available in flex container
  • flex: shorthand for flex-grow, flex-shrink and flex-basis
  • order: order in which to lay out an item in a flex container. Items are laid out in ascending order value. Defaults to source code order
  • align-self: overrides align-items for a specific item. Aligns item on cross axis
  • justify-self: Ignored in flexbox
