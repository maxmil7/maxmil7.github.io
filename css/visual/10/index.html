10. Flex

dislpay: flex is used to control the flow of items within a parent container.
A flex container has a start and an end and two main axis around which elements are laid out (main axis X, cross axis Y)
Parent can have multiple rows and columns
Properties are available on both parent and children to control the flow

10.1. Flext container properties

  • flex-direction: sets direction of flow of items
  • flext-wrap: determines if elements are wrapped when parent container runs out of space
  • flex-flow: shorthand for direction and flow
  • justify-content: controls content width and gaps along the main axis (horizontal axis)
  • align-items: controls alignment of items along the main axis
  • align-content: controls content width and gaps along the cross axis (vertical axis)
