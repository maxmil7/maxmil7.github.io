3.1. CSS box model

  • Each element is represented as a box by browser's rendering engine
  • The standard is called CSS box model
  • It determines the size, position and properties of these boxes

Content, Padding, Border, Margin

3.2. box-sizing

Sets how the total height and width of the element is calculated

3.2.1 content-box (default)

Height and width include content area only. Padding, margin and border add to the height and width
Default for HTML elements

3.2.2 border-box

Height and width include content, padding and border. Margin adds to the height and width
Default for CSS grid

3.2.2 padding-box

Height and width include content and padding. Margin and border add to the height and width
