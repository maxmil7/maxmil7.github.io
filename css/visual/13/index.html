13. CSS pre-processors

CSS pre-processors add new features to the syntax of CSS such as loops, conditionals, imports, functions etc.
They are a superset of CSS language and need to be compiled to CSS for use in browsers.

There are five CSS pre-processors Sass, SCSS,Less, Stylus and PostCSS

SCSS

SCSS language adds new features, @rules and global functions to shorten the amount of time spent in writing CSS

features
  • Variables: Declared using $ sign as opposed to --var $number: 1
  • Nested rules: Nested elements in CSS are accessed via space character. In SCSS they can be chained.
    #A { color: red; #B { color:green; } }
  • & character: Converted to name of parent element
    a { font-weight: bold; { &:hover { color: red } }
  • arithmetic operators: +, -, *, /, % instead of calc(). Also supports mod (%)
    div { width: 5px + (10px/2) * 3 - 1px
    height: 120px % 100 }
  • comparison operators: ==, !=, >, <, >=, <=
    @if ($padding > $margin) {
    padding: $padding }
    @else { padding: $margin }
  • String addition: String addition is supported as long as added string is trailing
    { p font: 50px Ari + "al";}
@rules
  • @use/@include loads mixins, functions, and variables from other SASS stylesheets or builtin sass modules
    @use 'styles/marketing';
  • @forward makes public members of loaded module available to end user as though they were defined in the file which loaded the module
    Functionality is not available for use within the module though (use @use instead)
    //bootstrap.scss
    @forward "src/list";
    //styles.scss
    @use "bootstrap";
    li { @include bootstrap.list-reset; }
  • @mixin/@include: used to define reusable pieces of styles
    @mixin reset-list {
    margin: 0;
    padding: 0;
    list-style: none; }
    nav ul { @include horizontal-list; }
  • @extend: used to inherit styles of another selector
    .error {
    border: 1px solid gray;
    &--serious {
    @extend .error;
    border-width: 2px; }
  • @function/@return: allows defining complex operations that can be reused
    @function timesTwo($val) {
    $result: $val*2;
    @return $result }
    .sidebar { margin-left: timesTwo(5) * 1px; }
  • @if <expression>{...}: controls whether block gets evaluated
    .test {
    $var: true;
    @if($var) { border: 1px solid gray; } }
  • @each <variable> in <expression> {...}: evaluated for each element of the list
    $classes: one, two, three;
    @each $class in $classes {
    .item-#{$class} { font-color: blue; } }
  • @for <variable> from <expression> to (or through) <expression> {...}: for loop
    @for $i from 1 to 3 {
    .item-#{$i} {
    font-color: blue; } }
  • @while <expression> {...}: evaluates block until expression returns true
    $i:1;
    @while $i < 3 {
    .item-#{$i} { font-color: blue; }
    $i: $i+1; }
