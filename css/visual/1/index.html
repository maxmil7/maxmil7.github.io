4. Assignment patterns

4.1. Shorthand

      //background
      background: background-color background-image background-repeat background-positon
      background: #000 url(images/bg.gif) no-repeat left top;

      //font
      font: font-style font-weight font-size line-height font-family
      font: italic bold .8em/1.2 Arial, sans-serif;

      //border
      border: border-width border-style border-color
      border: 1px solid #000

      //margin and padding
      margin: margin-top margin-right margin-bottom margin-left
      margin: 10px 5px 10px 5px;

      //border-radius
      //can specify two values vertical and horizontal
      border-radius: border-top-left-radius border-top-right-radius border-bottom-right-radius border-bottom-left-radius
      border-radius: 4px 2px 3px 4px
      border-radius: border-radius: 4px 3px 6px / 2px 4px;
      //Equivalent to T: 4px 2px R: 3px 4px B: 6px 2px L: 3px 4px

      //all - resets all element properties. Possible values: unset, initial, inherit

5.3. Pre-processors

    
Pre-processors add new features (eg. looping, conditions etc) not available in CSS
    Usually a superset of standard CSS features
    Examples include SASS, LESS

    
      $sizes: 40px, 50px, 80px;
      @each $size in $sizes {
        .icon-#{$size} {
          font-size: $size;
          height: $size;
          width: $size;
        }
      }
