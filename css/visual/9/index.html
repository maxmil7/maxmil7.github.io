9. Transformation

Elements can be transformed using the translate, scale and rotate properties. Transformations can be applied in both 2D and 3D.

9.1. 2D transformations

Translate: Moves the object on X and Y axis
Scale: Scales an element
Rotate: Rotates the element around origin (default center). Origin can be changed using the transform-origin property
Skew:Skews the element around X and Y axis
Matrix: CSS matrix can be used to achive the same effect as the above translations.
Syntax: transform: matrix(a, b, c, d, e, f)
ad scale, bc skew, ef translate (rotation can be achieved w/ a combination of these.
