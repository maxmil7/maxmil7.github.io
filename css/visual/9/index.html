9. Transformation

Elements can be transformed using the translate, scale and rotate properties. Transformations can be applied in both 2D and 3D.

9.1. 2D transformations

Translate

Moves the object on X and Y axis


Scale

Scales the elements

Example

Rotate

Rotates the element around origin (default center). Origin can be changed using the transform-origin property
