11. Grid

dislpay: grid is used to divide a page into multiple regions based on rows and columns.
Grid properties define the postion, size and gaps and layering between the regions.
Grid is automatically extended based on the regions within a grid.

11.1. Rows and columns within grid

  • grid-template-columns: definies column size within the grid
  • grid-template-rows: defines row size within grid
  • grid-auto-columns: defines column size for implicit (automatic) columns
  • grid-auto-rows: defines row size for implicit (automatic) rows

11.2. Template areas that span multiple columns and row

  • grid-column-start: start position within grid column
  • grid-column-end: end position within grid column
