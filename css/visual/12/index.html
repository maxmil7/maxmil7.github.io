11. Animation

Any element whose position, dimensions, angle or color can be changed can be animated. Animation is implemented using @keyframes at-rule and animation property.

  • Keyframes define the start (%) and end (%) state of an animated element
  • Animation property shorthand controls name, duration, easing, delay, count, direction, fill-mode and state of animation

@keyframes font { 0% { margin-top: 20px; } 100% { margin-top: 90px; font-size: 20px; color: indianred; } }
font 2s cubic-bezier(0.25, 0.1, 0.78, 1) 1s 2;

Animation properites

  • animation-name: name of keyframe specified by @keyframes directive
  • animation-duration: duration of animation in milliseconds or seconds
  • animation-timing-function: easing function defined by Bezier curve or steps
    Bezier curve (time X asis, velocity Y axis)takes x, y coordinates of 2 points along the curve
    Predifined values for bezier curve include ease, ease-in, ease-out etc.
    Steps(n, jumpterm) define number of stops along the transition points. Jumpterm specifies wherther to skip start, end etc.
  • animation-delay: delay before animation start in milliseconds or seconds
  • animation-iteration-count: number of times animation should play
  • animation-direction: normal, reverse, alternate or alternate-reverse animation direction
  • animation-fill-mode: state of animation when not playing forwards, backwards, none, both
  • animation-play-mode: paused, running (whether animation is paused or running)

Transition

Transition is used to define the transition between two states of elements specified.
States can be changed via CSS properties, pseudo-classes or via Javascript
Shorthand for property name, duration, easing function and delay
Transactions can be specified for multiple properties by providing comma separated values.

Transition properites

  • transition-property: CSS properties to which transition effect should be applied (margin, all, none etc.)
  • transition-duration: Duration of transition
  • transition-timing-function: Easing function similar to animation
  • transition-delay: Delay before transition is applied
