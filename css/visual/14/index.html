CSS at rules

At rules are CSS statements instructing CSS how to behave. They start with @. Common at rules include:

  • @import: used to import style rules from other style sheets. Optionally followed by media query for conditional imports. @import url('print.css') print;
  • @media: At rule that will apply its content if device meets criteria defined using media query @media screen and (min-width: 900px) {
    article { padding: 1rem; } }
  • @supports: At rule that will apply its content if browser supports given feature @supports (display: flex) {
    article { display: flex } }
  • @keyframes: Defines steps in a CSS animation sequence
    • @keyframes slidein {
    0% { transform: translateX(0%); }
    100% { transform: translateX(100%); } }
  • @page: Used to modify some CSS properties like margins, page-breaks etc. while printing a page @page { margin: 1cm; }

CSS global functions

Common CSS global functions include:

  • url(): Used to include a file background-image: url('https://abc.com')
  • attr(attr-name type, fallback): Used to retrieve value of an attribute of selected element and use it attr(data-count number, 0);
  • calc(): Used to perform calculations width: calc(100% - 80px)
  • minmax(min, max): Used in CSS grids, defines a size range >= min and < =max minmax(200px, 1fr)
  • var(custom value): Used to insert value of a custom property :root { --main-bg-color: coral; }
    div { background-color: var(--main-bg-color)}
  • contrast(value): Adjusts contrast of input image filter: contrast(1)
  • translate(x, y): Repositions element in horizontal or vertical directions transform: translate(100px, 200px)
  • rotate(angle): Rotates element around a fixed point transform: rotate(45deg)
  • scale(x-scale, y-scale): Resizes element on 2d plane transform: scale(0.7)
  • skew(x-angle, y-angle): Skews an element on 2d plane transform: skew(15deg, 15deg)
