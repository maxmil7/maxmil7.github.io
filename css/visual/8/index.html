8. Backgrounds

A variety of backgrounds can be created using gradients and images and their properties

8.1. Gradients

Possible types of gradients include: linear-gradient, radial-gradient, repeating-linear-gradient, repating-radial-gradient

Signature - background: type(rotation, color 1 px, color 2 px etc)

Rotation: degree, direction
Color: RGB, HCL, names
Length: %, pixels

8.2. Filters

Filters can be applied to any image or HTML element that produces graphic output
Signature - filter: type(value)
Values can be specified in percent, number, px or degree depending on type of filter

Fiters specified here because it is possible to set image as a background using background: url() or background-image: url() property
